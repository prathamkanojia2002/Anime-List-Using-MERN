### Setup .env file
## to run this local pc do these steps 
## step 1 create .env file 
```shell
MONGO_URI=your_mongo_uri
PORT=5000
```

### Step 2 Run this app locally

```shell
npm run build
```

### Step 3 Start the app

```shell
npm run dev
```
